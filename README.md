# 🌴 Atrativos Turísticos – Praia Grande/SP

Projeto acadêmico desenvolvido para apresentação dos principais atrativos turísticos da cidade de Praia Grande – São Paulo.
Autor
Regivan Santos Cruz

Ativida da disciplina de desenvolvimento de Sistemas – Desenvolvimento WEB
Referências

Prefeitura Municipal de Praia Grande
Exército Brasileiro
Governo do Estado de São Paulo

Uso de Inteligência Artificial
Este projeto utilizou apoio da IA ChatGPT (OpenAI) para auxílio na estruturação do código HTML, CSS e organização visual do layout.

O objetivo do projeto é demonstrar conhecimentos básicos em HTML e CSS através da construção de uma página WEB contendo informações turísticas organizadas em colunas e elementos visuais.

---

## 📌 Funcionalidades

✔ Página responsiva

✔ Layout em colunas

✔ Seis atrativos turísticos

✔ Efeito hover nas imagens

✔ Organização por cartões (cards)

✔ Referências das imagens

✔ Estrutura pronta para GitHub
