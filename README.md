# Atrativos_turísticos_em_Praia_Grande/SP

Projeto desenvolvido para apresentação dos principais atrativos turísticos da cidade de Praia Grande – São Paulo, como parte da Atividade do Curso Técnico em Desenvolvimento de Sistemas.

O objetivo é demonstrar conhecimentos básicos em HTML e CSS através da construção de uma página WEB contendo informações turísticas organizadas em colunas e elementos visuais.

## 📌 Funcionalidades trabalhadas:

✔ Página responsiva
✔ Layout em colunas
✔ Seis atrativos turísticos
✔ Efeito hover nas imagens
✔ Organização por cartões (cards)
✔ Referências das imagens
---

## 🏖 Atrativos apresentados
- Calçadão da Orla
- Portinho
- Fortaleza de Itaipu
- Parque Estadual Xixová
- Praça das Cabeças
- Vila Caiçara
---
## 🛠 Tecnologias utilizadas
